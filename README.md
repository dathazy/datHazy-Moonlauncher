Moonlight companion program that launches to datHazy UMVC3 Moonlight servers with support to launch through browser links.

Run MoonlauncherInstaller.exe to install. Each station has a "User Settings" and "Recommended Settings" option to connect. User Settings will respect your currently installed Moonlight's settings. Recommended Settings will auto-apply all recommended settings for optimal gameplay on connecting. Only use the "Set Moonlight Path" button if you have installed Moonlight in a custom location.

Includes an AutoHotKey script ZIP option for users that are more security-conscious and would not like to install a program. Do not download the ZIP if you are installing MoonlauncherInstaller.exe. Each script is only a few lines, making it super easy to audit the code.
Just unzip, double-click on the folder for "Lobby" or "Tournament" depending on whether a Lobby or Tournament is taking place, then double-click on the station you need to connect to with the same "User Settings" or "Recommended Settings" options described above.
